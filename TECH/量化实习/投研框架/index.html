    私募量化投研框架知识文档

    版本：V1.0

    核心思想：

    数据 → Alpha（因子）→ Alpha组合 → 组合优化 → 交易执行 → 风控归因 → 持续迭代

    一、整体框架

    现代量化私募本质上是一套工业化的投研体系，而不是依赖某一个"神奇策略"。

    整体流程如下：

                    市场数据
                    │
        ┌───────────┴────────────┐
        │                        │
    数据研究(Data)          另类数据
        │
        ▼
    因子研究(Alpha Research)
        │
        ▼
    组合研究(Portfolio Research)
        │
        ▼
    优化器(Portfolio Optimizer)
        │
        ▼
    算法交易(Execution)
        │
        ▼
    风控(Risk Control)
        │
        ▼
    收益归因(Attribution)
        │
        ▼
    持续迭代(Research Loop)

    整个流程每天不断循环。

    二、数据研究（Data Research）

    目标

    数据是量化投资的基础。

    没有高质量的数据，就没有高质量的Alpha。

    一句话：

    Garbage In, Garbage Out.

    数据来源

    1、行情数据（Market Data）

    包括：

    • Tick数据
    • Level2
    • 分钟线
    • 日线
    • 成交量
    • 成交额

    例如：

    Open
High
Low
Close
Volume
Amount

    2、基本面数据（Fundamental Data）

    例如：

    • ROE
    • EPS
    • 营收
    • 净利润
    • 毛利率
    • 资产负债率

    主要来源：

    • Wind
    • Choice
    • CSMAR

    3、另类数据（Alternative Data）

    近年来越来越重要。

    例如：

    • 新闻
    • 公告
    • 社交媒体
    • 舆情
    • 招聘数据
    • 卫星图片
    • 电商销量
    • 专利数据
    • ESG

    4、高频数据（High Frequency Data）

    主要用于：

    • 高频交易(HFT)
    • 做市(Market Making)
    • 统计套利(Stat Arb)

    例如：

    Bid1
Ask1
Bid2
Ask2
...

    以及：

    • 逐笔成交
    • 逐笔委托
    • 撤单数据

    • 幸存者偏差

    三、因子研究（Alpha Research）

    什么是Alpha？

    Alpha：

    能够预测未来收益率的信号。

    量化研究员每天最核心的工作就是：

    不断寻找新的Alpha。

    常见Alpha

    Momentum（动量）

    过去涨得多：

    未来继续上涨。

    例如：

    Return(20)

    Reversal（反转）

    跌太多：

    未来反弹。

    例如：

    -Return(5)

    Value（价值）

    例如：

    • PE
    • PB
    • EV/EBITDA

    估值越低越好。

    Quality（质量）

    例如：

    • ROE
    • 毛利率
    • 自由现金流

    Growth（成长）

    例如：

    • 营收增长
    • 利润增长
    • 研发投入

    高频Alpha

    例如：

    Order Imbalance：

    Buy Volume
-
Sell Volume

    还有：

    • Queue Imbalance
    • Cancel Ratio
    • Micro Price

    四、因子验证（Factor Research）

    提出Alpha以后，需要验证它是否真正有效。

    IC（Information Coefficient）

    衡量：

    因子值

    与

    未来收益率

    之间的相关性。

    例如：

    IC = 0.05

    股票领域：

    IC > 0.03

    已经属于不错的因子。

    Rank IC

    股票多使用Rank IC。

    原因：

    排序比数值更稳定。

    ICIR

    稳定性指标：

    ICIR

=

Mean(IC)

/

Std(IC)

    越高越稳定。

    分层测试（Layer Test）

    按照因子排序：

    Top10%

...

Bottom10%

    观察：

    Top组合是否持续跑赢Bottom组合。

    Turnover

    换手率。

    换手越高：

    交易成本越高。

    Alpha Decay

    Alpha衰减速度。

    例如：

    预测未来：

    • 1天
    • 3天
    • 5天
    • 10天

    还能否保持预测能力。

    五、组合研究（Portfolio Research）

    真实私募：

    不会只依赖一个Alpha。

    通常：

    Alpha1

Alpha2

...

Alpha1000

    共同工作。

    例如：

    Momentum

Value

Quality

Growth

Sentiment

News

Liquidity

Volatility

    Alpha融合

    例如：

    某股票：

    Momentum

0.8

0.4

-0.2

1.3

    最终：

    Final Alpha

=

0.3×Momentum

+

0.2×Quality

+

0.4×Sentiment

+

0.1×Value

    称为：

    Composite Alpha。

    深度学习融合

    现代私募越来越多使用：

    • XGBoost
    • LightGBM
    • LSTM
    • Transformer

    自动学习：

    哪些Alpha近期更有效。

    六、优化器（Portfolio Optimizer）

    优化器负责：

    把预测结果转换成最终仓位。

    例如：

    预测：

    贵州茅台

10%

五粮液

8%

泸州老窖

7%

    全部都是白酒。

    风险过于集中。

    因此：

    需要优化。

    优化目标

    通常：

    Max

Expected Return

-

Risk

-

Transaction Cost

    即：

    收益最大

    风险最小

    成本最低。

    常见约束

    例如：

    行业暴露

市值暴露

Beta

波动率

单票权重

成交额

换手率

杠杆

    最终输出：

    Target Portfolio

    七、算法交易（Execution）

    理论收益 ≠ 实盘收益。

    真正赚钱：

    靠执行。

    例如：

    买100万股

    不能：

    一次成交。

    否则：

    价格会被自己拉高。

    因此：

    拆单：

    100万股

↓

1000次

↓

每次1000股

    常见算法

    TWAP

    时间均匀交易。

    VWAP

    成交量加权交易。

    POV

    根据市场成交量比例交易。

    Implementation Shortfall

    降低市场冲击成本。

    Smart Order Routing

    智能订单路由。

    T0策略

    这里的"T0"并不是所有A股当天买卖。

    通常包括：

    • ETF T+0
    • 可转债
    • 股指期货
    • 底仓做T
    • 融券

    目的：

    提高收益。

    目标：

    真正赚Alpha的钱。

    而不是行业Beta的钱。

    实时监控：

    单票仓位

PnL

+0.2%

-0.1%

-0.3%

+1.0%

    分析：

    哪些Alpha有效。

    哪些已经失效。

    十五、核心思想总结

    现代量化私募真正的竞争力来自于：

    工业化投研体系。

    包括：

    • 数据工程
    • Alpha研发
    • Alpha融合
    • 风险模型
    • 组合优化
    • 算法交易
    • 风控系统
    • 持续迭代

    而不是依赖某一个"神奇因子"。

    一句话总结

    现代量化私募的本质，是利用数据工程构建高质量数据平台，通过持续研发Alpha信号，结合风险模型和组合优化形成投资组合，再利用算法交易高效执行，并通过收益归因不断迭代策略，最终形成可持续、可复制的量化投研闭环。
