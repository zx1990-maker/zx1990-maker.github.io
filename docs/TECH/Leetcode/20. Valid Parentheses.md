# 20. Valid Parentheses

## 题目描述：
Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

An input string is valid if:

Open brackets must be closed by the same type of brackets.
Open brackets must be closed in the correct order.
Every close bracket has a corresponding open bracket of the same type.

Example 1:

> Input: s = "()"

> Output: true

Example 2:

> Input: s = "()[]{}"

> Output: true

Example 3:

> Input: s = "(]"

> Output: false

Example 4:

> Input: s = "([])"

> Output: true

Example 5:

> Input: s = "([)]"

> Output: false


***


## Solution:
```python
class Solution(object):
    def isValid(self, s):
        stack = []
        # 建立右括号到左括号的映射
        mapping = {'}': '{', ')': '(', ']': '['}
        for i in s:
            # 如果是右括号
            if i in mapping:  
                # 栈不为空则弹出栈顶，否则给个无法匹配的占位符 '#'
                top = stack.pop() if stack else '#'
                # 检查弹出的左括号是否与当前右括号匹配
                if top != mapping[i]:
                    return False
            # 如果是左括号，入栈
            else:  
                stack.append(i)     
        # 检查栈是否为空（确保没有多余的左括号）
        return len(stack) == 0
```
这道题是栈（Stack）的最经典应用。栈的特点是 后进先出（LIFO - Last In First Out）。 为什么栈完美契合这道题？**因为当我们遇到一个右括号时，我们需要找的是“最近出现的左括号”，而“最近出现”正对应着“最后进栈”的元素。**

算法执行流程（剥洋葱模型）：

1. 准备一个空栈。
2. 从头到尾逐个字符读取文件内容。
3. 如果看到的是左括号（`(`、`[`、`{`），不用多想，直接压入栈中（Push）。
4. 如果看到的是右括号（`)`、`]`、`}`）：
    * 首先检查栈是否为空。如果栈已经空了，说明这个右括号孤苦伶仃，没有左括号能和它配对，直接判定非法，返回 `False`。
    * 如果栈不为空，从栈顶弹出（Pop）一个左括号，检查这两个括号是不是同一种类型：
        * 如果匹配（比如弹出的是 `(`，当前是 `)`），那就安全，继续看下一个字符。
        * 如果不匹配（比如弹出的是 `[`，当前是 `)`），直接判定非法，返回 `False`。
5. 当文件所有的字符都读完后，做最后一步检查：看栈是不是空的。
    * 如果栈空了，说明所有左括号都有了归宿，返回 `True`。
    * 如果栈没空（里面还剩有左括号），说明有左括号落单了（比如输入是 `{[()`），判定非法，返回 `False`。
