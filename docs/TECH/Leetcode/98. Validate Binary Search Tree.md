# 98. Validate Binary Search Tree

**题目链接**：[LeetCode - Validate Binary Search Tree](https://leetcode.com/problems/validate-binary-search-tree/description/)

## 题目描述

Given the root of a binary tree, determine if it is a valid binary search tree (BST).

A valid BST is defined as follows:
* The left subtree of a node contains only nodes with keys **strictly less than** the node's key.
* The right subtree of a node contains only nodes with keys **strictly greater than** the node's key.
* Both the left and right subtrees must also be binary search trees.

---

## Solution:

该题利用了 **BST (二叉搜索树) 中序遍历结果为严格升序** 的特性。通过中序遍历（左 -> 根 -> 右）并使用一个全局/类变量 `prev` 记录前一个节点的值，如果当前节点的值小于或等于 `prev`，则说明不是合法的 BST。

```python
# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution(object):
    def isValidBST(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: bool
        """
        self.prev = float('-inf')
        
        def inorder(root):
            if root is None:
                return True
                
            # 1. 递归检查左子树
            if not inorder(root.left):
                return False
                
            # 2. 检查当前节点是否满足严格升序
            if self.prev >= root.val:
                return False
            self.prev = root.val  # 更新前一个节点的值
            
            # 3. 递归检查右子树
            return inorder(root.right)
            
        return inorder(root)
```
#### Q：为什么不用验证self.val大于右子树的值？
你可以把 self.prev 想象成一个在有序数组上不断向右移动的滑块。

1. 打印完左子树的所有节点。此时，self.prev 停留在左子树的最大值上。

2. 遇到根节点（root）。代码检查 root.val <= self.prev。这就验证了：根节点 > 左子树的最大值。

3. 关键点来了：验证完根节点后，执行了 self.prev = root.val。此时，self.prev 变成了根节点的值。

4. 紧接着，递归进入右子树。右子树里碰到的第一个节点（右子树里最左边的那个，即右子树的最小值），它在进行检查时，它的 self.prev 是谁？刚好就是刚才存进去的根节点的值！

5. 代码执行 右子树节点.val <= self.prev(根节点)。这就验证了：右子树的节点 > 根节点。
