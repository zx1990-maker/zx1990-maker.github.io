# 98. Validate Binary Search Tree

**题目链接**：[LeetCode - Validate Binary Search Tree](https://leetcode.com/problems/validate-binary-search-tree/description/)

## 题目描述

Given the root of a binary tree, determine if it is a valid binary search tree (BST).

A valid BST is defined as follows:
* The left subtree of a node contains only nodes with keys **strictly less than** the node's key.
* The right subtree of a node contains only nodes with keys **strictly greater than** the node's key.
* Both the left and right subtrees must also be binary search trees.

---

## Solution:

该题利用了 **BST (二叉搜索树) 中序遍历结果为严格升序** 的特性。通过中序遍历（左 -> 根 -> 右）并使用一个全局/类变量 `prev` 记录前一个节点的值，如果当前节点的值小于或等于 `prev`，则说明不是合法的 BST。

```python
# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution(object):
    def isValidBST(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: bool
        """
        self.prev = float('-inf')
        
        def inorder(root):
            if root is None:
                return True
                
            # 1. 递归检查左子树
            if not inorder(root.left):
                return False
                
            # 2. 检查当前节点是否满足严格升序
            if self.prev >= root.val:
                return False
            self.prev = root.val  # 更新前一个节点的值
            
            # 3. 递归检查右子树
            return inorder(root.right)
            
        return inorder(root)
