# 701. Insert into a Binary Search Tree
https://leetcode.com/problems/insert-into-a-binary-search-tree/
## 题目描述：
You are given the root node of a binary search tree (BST) and a value to insert into the tree. Return the root node of the BST after the insertion. It is guaranteed that the new value does not exist in the original BST.

Notice that there may exist multiple valid ways for the insertion, as long as the tree remains a BST after insertion. You can return any of them.

## Solution：
```Python
class Solution(object):
    def insertIntoBST(self, root, val):
        """
        :type root: Optional[TreeNode]
        :type val: int
        :rtype: Optional[TreeNode]
        """
        #这个函数返回在root子树插入val的root
        if root==None:
            node=TreeNode(val)
            return node
        
        if val<root.val:
            root.left = self.insertIntoBST(root.left,val)
        if val>root.val:
            root.right = self.insertIntoBST(root.right,val)    
        return root
```
优化思维（完全信任子问题）：你坐在当前 root 的办公室里，不需要去替下一层判断它是不是 None。

如果 val < root.val，你直接对左手下说：“拿着这个 val，去我的左子树里把它插好，插好之后把左子树新的根节点还给我。”

下属（递归）执行完后，把结果交到你手里，你只需要用 root.left = ... 接住就行了！

利用子问题的返回值，我们不再需要在当前层写任何 if root.left == None 这样的臃肿判断
