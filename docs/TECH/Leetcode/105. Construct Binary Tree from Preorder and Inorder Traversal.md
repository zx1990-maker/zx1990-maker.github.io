# 105. Construct Binary Tree from Preorder and Inorder Traversal
**题目链接：** https://leetcode.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal/description/
## 题目描述：
Given two integer arrays preorder and inorder where preorder is the preorder traversal of a binary tree and inorder is the inorder traversal of the same tree, construct and return the binary tree.

## Solution：
```Python
class Solution(object):
    def buildTree(self, preorder, inorder):
        """
        要素一（语义）：给它两段数组，它承诺把建好的子树根节点返回回来。
        """
        # 要素二：Base Case
        # 如果数组为空，说明分无可分，撞到了空位，返回 None
        if not preorder or not inorder:
            return None
        
        # 要素三：当前层合并逻辑
        # 1. 揪出根节点
        root_val = preorder[0]
        root = TreeNode(root_val)
        
        # 2. 定位黄金分割点：根节点在中序遍历中的位置
        i = inorder.index(root_val)
        
        # 3. 信任子问题 + 降维打击（精确切片委派任务）
        root.left = self.buildTree(preorder[1 : 1+i], inorder[0 : i])
        root.right = self.buildTree(preorder[1+i : ], inorder[i+1 : ])
        
        # 4. 履行契约：把今年重构好的总根节点向上交差
        return root
```
