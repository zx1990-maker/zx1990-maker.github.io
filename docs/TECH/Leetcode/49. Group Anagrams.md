# 49. Group Anagrams

**Links:** https://leetcode.com/problems/group-anagrams/description/

## Description:

Given an array of strings strs, group the anagrams together. You can return the answer in any order.

## Solution:

```Python
class Solution(object):
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        memo = {} # 原生字典
        
        for word in strs:
            # 1. 统计频次
            count = [0] * 26
            for char in word:
                count[ord(char) - ord('a')] += 1
                
            # 2. 转换为元组作为 Key
            # Python 的字典严禁使用“可变（Mutable）”的数据结构作为 Key，必须使用“不可变（Immutable）”的数据结构。
            key = tuple(count)
            
            # 3. 原生字典的 if-else 处理
            if key in memo:
                memo[key].append(word)
            else:
                memo[key] = [word]
                
        return list(memo.values())
```
