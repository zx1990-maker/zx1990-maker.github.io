# 236. Lowest Common Ancestor of a Binary Tree

**题目链接**：https://leetcode.com/problems/lowest-common-ancestor-of-a-binary-tree/description/

## 题目描述

Given a binary tree, find the lowest common ancestor (LCA) of two given nodes in the tree.

According to the definition of LCA on Wikipedia: “The lowest common ancestor is defined between two nodes p and q as the lowest node in T that has both p and q as descendants (where we allow a node to be a descendant of itself).”

## Solution:

```python
class Solution(object):
    def lowestCommonAncestor(self, root, p, q):
        """
        ===================================================================
        1. 递归函数输入与输出的定义 (Function Semantics)
        ===================================================================
        输入：当前子树的根节点 root
        输出 (通过 return 返回)：
             - 若子树中包含 p 或 q，则返回那个被找到的目标节点。
             - 若子树中同时包含 p 和 q，则返回它们计算出来的最近公共祖先 (LCA)。
             - 若子树中既没有 p 也没有 q，则返回 None。
        """

        # ===================================================================
        # 2. Base Case 是什么？(安全出口 & 触底信号)
        # ===================================================================
        # - 情况 A：root == None，说明搜索到了叶子节点的下方，直接返回 None。
        # - 情况 B：root == p 或 root == q，说明在当前节点撞到了目标。
        #   此时不需要继续向下搜索，直接把当前节点作为“已找到目标”的信号向上输出。
        if root == None or root == p or root == q:
            return root
        
        # ===================================================================
        # 3. 递归逻辑：怎么利用子问题的返回值来搭建当前层的逻辑？
        # ===================================================================
        
        # 【分】让左右子树去干活，并用极具逻辑性的变量名接住它们的“成果汇报”
        left_return = self.lowestCommonAncestor(root.left, p, q)
        right_return = self.lowestCommonAncestor(root.right, p, q)

        # 【总】根据左右子树汇报上来的 left_return 和 right_return，在当前层做分类决策：
        
        # 决策一：如果左子树汇报说“没找到任何目标(None)”
        # 那么最终结果完全取决于右子树的汇报。
        # (若右子树找到了目标/LCA，则返回右子树的结果；若右子树也是 None，这里也天然返回了 None)
        if left_return == None:
            return right_return
            
        # 决策二：如果右子树汇报说“没找到任何目标(None)”
        # 那么同理，直接把左子树找到的有效结果向上交差。
        elif right_return == None:
            return left_return
            
        # 决策三：如果左右子树都传回了非空的节点 (left_return 和 right_return 都有值)
        # 严格意味着：目标 p 和 q 分别落在了当前 root 节点的左右两侧。
        # 根据定义，当前 root 就是隔离两者的那个最低点，即最近公共祖先。因此直接返回 root。
        elif left_return and right_return:
            return root
```
