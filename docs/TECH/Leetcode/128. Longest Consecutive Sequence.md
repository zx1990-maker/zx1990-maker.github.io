## Description:

Given an unsorted array of integers nums, return the length of the longest consecutive elements sequence.

You must write an algorithm that runs in O(n) time.
<img width="853" height="306" alt="image" src="https://github.com/user-attachments/assets/900e6dbb-5ceb-4d52-8bad-b3ffb263f33e" />


## Solution:

```Python
class Solution(object):
    def longestConsecutive(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        # 边界情况拦截：如果是空数组，直接返回 0
        if not nums:
            return 0
            
        # 1. 全部扔进集合，去重且支持 O(1) 查找
        num_set = set(nums)
        longest_streak = 0
        
        # 2. 遍历集合中的每一个数字
        for num in num_set:
            # 3. 核心逻辑：判断当前数字是不是一个序列的“车头”
            # 如果 num - 1 不在集合中，说明它是起点，我们才开始往后数
            if (num - 1) not in num_set:
                current_num = num
                current_streak = 1
                
                # 只要 num + 1 还在集合里，就一直顺藤摸瓜找下去
                while (current_num + 1) in num_set:
                    current_num += 1
                    current_streak += 1
                    
                # 更新历史最长记录
                longest_streak = max(longest_streak, current_streak)
                
        return longest_streak


```
